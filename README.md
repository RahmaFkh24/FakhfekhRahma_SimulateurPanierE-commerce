# FakhfekhRahma_SimulateurPanierE-commerce
# 🛒 Simulateur de panier e-commerce

## 📌 Description du projet

Ce projet est une application web interactive permettant de simuler un panier d’achat en ligne.
L’utilisateur peut ajouter des produits, modifier leur quantité, appliquer un code promo et voir le total se mettre à jour en temps réel.

L’objectif est de mettre en pratique les bases du développement front-end (HTML, CSS, JavaScript) tout en offrant une expérience utilisateur fluide et moderne.

---

## 🛠️ Technologies utilisées

* HTML5 (structure de la page)
* CSS3 (design, animations et responsive)
* JavaScript (logique, interactions, manipulation du DOM)
* LocalStorage (sauvegarde des données côté navigateur)

---

## ✨ Fonctionnalités principales

* ➕ Ajouter des produits au panier
* ➖ Modifier la quantité des produits (+ / -)
* 🗑️ Supprimer un produit du panier
* 💰 Calcul automatique du total
* 🎟️ Application d’un code promo (-10%)
* 📦 Affichage du nombre total d’articles (badge)
* 💾 Sauvegarde du panier avec localStorage
* 🧾 Message “panier vide”
* 🖼️ Affichage des produits avec images
* 🎨 Interface moderne avec animation de fond

## 🚀 Nouveautés explorées

* Utilisation de **localStorage** pour sauvegarder les données utilisateur
* Manipulation avancée du **DOM en JavaScript**
* Création d’animations CSS (hover, transitions, background animé)
* Gestion d’un état dynamique (panier avec quantités)
* Amélioration de l’expérience utilisateur (UX/UI)

---

## ⚠️ Difficultés rencontrées

* Gestion des quantités des produits sans créer de doublons
* Mise à jour dynamique du DOM après chaque action
* Calcul correct du total avec le code promo
* Sauvegarde et récupération des données avec localStorage

---

## ✅ Solutions apportées

* Utilisation de **tableaux JavaScript** pour gérer les produits
* Utilisation de la méthode `find()` pour vérifier l’existence d’un produit
* Création de fonctions dédiées pour mettre à jour le panier (`updateCart`)
* Utilisation de **localStorage** avec `JSON.stringify` et `JSON.parse`
* Tests progressifs et correction des bugs étape par étape
